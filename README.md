# Thjalfi

Thhjalfi is CLI gem generator with Thor.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'thjalfi'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install thjalfi
